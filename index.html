Developer Guide

Permite mostrar uma notificação de acordo com o status do conteúdo como previamente configurado na página de Conteúdos do MyEduzz.


Função "notificationCheckoutEduzz"

Deve ser implementado junto ao id do conteúdo.


É necessário implementar o código no final da sua página HTML como no exemplo: 

<!DOCTYPE html>
<html>
	<head>
		...
		<link rel="stylesheet" type="text/css" href="http://eduzz.com/dev/js/notification-checkout-eduzz.min.css">
		
		...
	</head>
	<body>
		...
		<script type="text/javascript" src="http://eduzz.com/dev/js/notification-checkout-eduzz.min.js"></script>
		<script type="text/javascript">
			notificationCheckoutEduzz.init(1 '1668');
		</script>
	</body>
</html>

1 ID do conteúdo referente ao cnt_id na tabela TEDConteudos @string
